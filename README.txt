Създаване на базата данни със статусите
========================================
редактираш init_database.rb и най-отдолу пишеш статусите - по един ред за всеки ден. редовете най-отдолу след __END__ това ще са вашите публикации.
след което пускаш init_database.rb

ruby init_database.rb

пускане
========================================
слагаш в yatoto.rb акаунтите и пускаш

ruby yatoto.rb
